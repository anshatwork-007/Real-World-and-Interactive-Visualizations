# Experiment-18  
## Title: Real-World and Interactive Visualizations  

---

## 🎯 Aim  
To study and implement real-world and interactive visualization techniques using Python libraries including Plotly.

---

## 🎯 Objectives  
- To understand advanced and interactive visualization methods  
- To visualize hierarchical, clustered, geographical, and multidimensional data  
- To analyze real-world datasets using interactive charts  
- To use Plotly and other Python libraries for dynamic visualizations  

---

## 💻 Software Required  
- Python  
- Jupyter Notebook / Google Colab  
- Libraries: Pandas, Matplotlib, Seaborn, Plotly, SciPy, WordCloud, Matplotlib-Venn  

---

## 📘 Theory  
Real-world and interactive visualizations are used to represent complex datasets in an engaging and meaningful way. These charts allow users to interact with data through zooming, hovering, filtering, and dynamic exploration.  

Plotly is a popular Python library used for creating highly interactive charts and dashboards. It supports web-based visualizations with hover labels, animations, and real-time interactivity.

---

## 📊 Visualization Techniques Studied  

### 1. Treemap  
Displays hierarchical data using nested rectangles.  

**Applications:**
- Department budget analysis  
- Product category sales  
- Disk storage usage  

👉 Plotly Support: Interactive treemaps with hover details  

---

### 2. Dendrogram  
Tree-like diagram used for hierarchical clustering.  

**Applications:**
- Student grouping  
- Customer segmentation  
- Similarity analysis  

---

### 3. Venn Diagram  
Shows common and unique elements between sets.  

**Applications:**
- Student subject combinations  
- Common interests analysis  

---

### 4. Sankey Diagram (Student Flow)  
Represents flow from one stage to another using arrows or links.  

**Applications:**
- Admission to placement flow  
- Course progression  
- Website navigation flow  

👉 Plotly Support: Interactive Sankey charts  

---

### 5. 3D Scatter Plot  
Displays relationship among three numerical variables.  

**Applications:**
- Income, loan amount, credit score  
- Study hours, marks, attendance  

👉 Plotly Support: Rotatable and zoomable 3D charts  

---

### 6. WordCloud  
Displays words where size represents frequency.  

**Applications:**
- Student feedback  
- Review analysis  
- Social media trends  

---

### 7. Geospatial Visualization  
Represents data on maps using latitude and longitude.  

**Applications:**
- Population mapping  
- Delivery tracking  
- Branch locations  

👉 Plotly Support: Interactive maps with zoom and hover  

---

## ⚡ Interactive Features of Plotly  
- Zoom in / Zoom out  
- Hover tooltips  
- Filtering and legends  
- Rotatable 3D graphs  
- Animated charts  
- Real-time dashboard support
